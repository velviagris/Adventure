# 🌍 世界冒险 (Adventure)

**世界冒险** 是一款基于地理位置的“世界迷雾”类足迹记录应用。它不仅能在后台默默记录你的生活轨迹，还能将你走过的每一个角落从“未知的迷雾”中点亮。无论是日常的城市通勤，还是跨越大洋的环球旅行，它都是你忠实的足迹编年史。

## 📖 简介

本应用将地球划分为无数个网格。随着你在现实世界中的移动，应用会实时解开地图上的迷雾，展现你真实的探索范围。应用内建了智能省电逻辑、精准的球面多边形面积计算、自动下载行政边界以及多维度的成就系统。

你产生的所有足迹数据均保留在本地，随时可导出/恢复备份。

---

## ✨ 亮点功能

### 🗺️ 双重精度探索系统
为了平衡电池消耗与记录精度，应用提供两种无缝切换的探索模式：
* **📍 高精度旅游模式 (Precise Mode)**：记录最精细的街道级网格（18级缩放），适合徒步漫游、景点打卡，让你成为真正的“像素猎人”。
* **🔋 省电日常模式 (Battery Mode)**：记录粗略的区/村级大网格（14级缩放），覆盖范围广且极度省电，适合高铁、自驾或日常通勤。

### 🔋 智能休眠与无感记录
* **Activity Recognition 深度整合**：应用能够智能识别你的运动状态。当手机处于“静止 (Still)”状态超过一定时间后，将自动暂停 GPS 定位，进入极致省电的休眠模式；一旦检测到移动（步行、骑行、乘车），立刻唤醒继续记录。

### 🏙️ 真实的行政区划与进度测算
* **真实边界解析**：应用会根据你的位置自动从 OpenStreetMap (Nominatim API) 下载所在市、省、国的真实 GeoJSON 边界。
* **精准面积计算**：摒弃粗略的 BBox（边界框）算法，采用了基于地球球面域的线积分求积公式 (Spherical Polygon Area Line Integral) 与射线法 (Ray-casting)，精确计算出你在各级行政区内的实际点亮面积与百分比。

### 🏆 丰富的趣味成就系统 (Achievement Engine)
探索世界不仅是记录，更是一场充满挑战的游戏。内置完善的离线评判引擎，包含数十种成就徽章：
* **硬核数据**：全球探索总面积（大地漫步者）、总移动距离（行者无疆）、踏足国家数（环球旅行家）。
* **趣味连胜**：连续探索天数（开拓狂热）、连续“零探索”天数（宅家隐士）、对单一网格的重复造访（故地重游）、单次最远跃迁距离（时空跃迁）。
* 成就分为 *铜、银、金、铂金、玛瑙* 五个阶梯，满足你的收集癖。

### 📊 每日探索简报
* 结合 `WorkManager`，每天晚上 21:00 准时为你推送今日战报（Daily Summary），总结一天内“深度探索”与“途经”的全新区域面积。

### 🗄️ 完全的数据主权与管理
* **本地优先**：所有足迹网格、成就、行政区数据均通过 Room Database 存储于本地。
* **全量导出/导入**：提供一键 JSON 导出与恢复功能。换手机或重置系统也能轻松携带你的“一生足迹”。
* **日志系统**：内建自定义崩溃拦截与日志系统 (`AppLogger`)，一键导出排查问题。

## 🛠️ 技术栈

本项目遵循现代 Android 开发的最佳实践与 Native 设计规范：

* **UI 框架**: Jetpack Compose + Material Design 3 (支持动态取色)
* **地图渲染**: MapLibre GL Native (矢量瓦片、高性能遮罩与图层堆叠)
* **架构模式**: MVVM (Model-View-ViewModel) + Coroutines/Flow 响应式数据流
* **本地存储**: Room Database (SQLite) + DataStore (Preferences)
* **后台服务**: Foreground Service + FusedLocationProviderClient + ActivityRecognitionClient
* **网络服务**: OkHttp (获取 OSM GeoJSON 边界数据)
* **任务调度**: WorkManager (每日总结后台任务)
